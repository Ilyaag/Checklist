4. Чек лист iOS guidelines, 15-20 кейсов -  https://developer.apple.com/design/human-interface-guidelines/ios/overview/themes/

1.	Использование пользователем всего дисплея
2.	Быстрые действия для важных задач
3.	Краткое название для каждого быстрого действия
4.	Смайлы для использования глифа не используются
5.	Ненавязчивый брендинг
6.	Брендинг не мешает дизайну приложения
7.	Приоритет контенту
8.	Нет отображения логотипа везде во всем приложении
9.	Внешний режим соответствует выбору юзера в настройках(например дарк мод)
10.	Читаемость контента в различных режимах
11.	Экран запуска идентичен начальному экрану вашего приложения
12.	Нет обилия текста на экране запуска
13.	Нет рекламы на экране запуска
14.	Все изображения в приложении предоставлены в высоком разрешении для всех устройств которые поддерживают данное приложение
15.	Используется сетка 8px-by-8px
16.	Иконка приложения простая, узнаваемая, углы значков прямые
17.	Масштабирование, прокрутка работает исправно
18.	Текст разборчивый
19.	Навигация вперед/назад включается когда это необходимо
20.	Избегание использования web вида через браузер
